How to play

The game offers both a command line interface and a graphical interface. A match can have from 3 to 6 players, human or CPU-controlled.
The game's rules details can be found at : http://www.goodlittlegames.co.uk/games/09-witch-hunt.html.

For developpers

This software was developed with the objectives of modularity and extensibility.
Thus, any addition of new cards, features or rules is welcome, as well as any improvement of already existing components.

The main method can be found at fr.sos.witchhunt.controller.Application.

Both HumanPlayer and CPUPlayer classes inherit from the abstract class Player.

To each Rumour Card corresponds a class of the package fr.sos.witchhunt.model.Cards.
